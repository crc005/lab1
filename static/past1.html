Every One Rich Mobile Banking - CSE110 Fall 2014

This is an Android application created for Every One Rich Bank with a team of undergraduates. The banking app allow their users to create a new user account, see their finanical statements, transfer/debit/credit money, login securely to their user account, and lets user create multiple bank accounts (eg. checking/saving) in their user account. Visit our github repository.
